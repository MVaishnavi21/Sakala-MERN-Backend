## Sakala MERN Challenge - Day 11 ✅

**Goal:** Connect React form to MongoDB with full validation

**Features Shipped:**
- React form: name, age, email inputs
- Express POST `/profile` route saves to MongoDB
- Mongoose schema with `required: true, unique: true` on email
- Frontend + Backend validation
- Duplicate email error handling: `E11000`
- GET `/users` returns all users from DB

**Tech Used:** React, Express, MongoDB, Mongoose, Vite, CORS

**How to run:**
Backend: `node index.js` on port 3000
Frontend: `npm run dev` on port 5173
